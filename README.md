# Module 8 Assignment: Dictionaries and Data Aggregation

## 📊 Overview
This project builds a simple **customer management system** for a fictional company called **GlobalTech Solutions**.

The goal is to use Python dictionaries and basic programming concepts to store, manage, and analyze customer and project data.

---

## 🧠 What This Project Does

- Stores service pricing using dictionaries
- Manages customer information (name, contact, email, phone)
- Tracks projects for each customer
- Calculates project costs based on hourly rates
- Performs basic data analysis and summaries
- Uses dictionary comprehensions for data transformations
- Generates reports and recommendations
